Pixieの古い(2010年くらい)レガシーのサンプルです。
Photonmapのサンプルです。
使用は自己責任でお願いします。

# Using Pixie Renderer 2.2.6

https://sourceforge.net/projects/pixie/

# 1. Compile shaders.
```
sdrc *.sl
```
# 2. Rendering
```
rndr pmap_render_pass0.rib
```
next
```
rndr pmap_render_pass1.rib
```
next
```
rndr pmap_render_pass2.rib
```
